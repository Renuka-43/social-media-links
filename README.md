# 🌐 Social Media Links

A simple and responsive **Social Media Links** webpage built using **HTML5** and **CSS3**. This project provides a clean profile card containing social media buttons that redirect users to different platforms with a modern and user-friendly interface.

---

## 📌 Features

- 🎨 Clean and Modern UI
- 📱 Responsive Design
- 🔗 Clickable Social Media Buttons
- ✨ Hover Effects and Smooth Transitions
- 💻 Lightweight and Easy to Customize
- 🌍 Compatible with All Modern Browsers

---

## 🛠️ Technologies Used

- HTML5
- CSS3
